# Centrala alarmowa

Projekt PCB wraz z oprogramowaniem mikrokotrolera PIC18F25K80 w <a href="https://www.ccsinfo.com/">CCS C</a>.
Komunikacja z <a href="https://github.com/marcin-filipiak/pic_alarmrfid">czytnikiem kart RFID</a> i innymi urządzeniami po magistrali CAN, 
obsługa czujników ruchu oraz <a href="https://github.com/marcin-filipiak/pic_alarmsirene">syreny alarmowej</a>.

Konfiguracja centrali przez UART.

Montaż na szynie DIN.

Zasilanie 230V.

Wejścia czujników ruchu itp. z optoizolacją.
